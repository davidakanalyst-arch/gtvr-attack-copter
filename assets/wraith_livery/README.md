# Wraith stealth livery artwork

`wraith_stealth_camo_v1.png` is the project-bound source pattern used by the dev build to generate the two exterior body atlases. The build preserves the original dark vents, cavities, soot and panel gaps, and it does not repaint the separate weapons, sensors or auxiliary-fixture atlas.

- Generation mode: built-in image generation
- Intended use: seamless, low-visibility military helicopter albedo pattern
- Integrated by: `tools/build_gtvr_wraith_dev.py`

## Generation prompt

> Use case: stylized-concept. Asset type: seamless tileable game texture for a military helicopter livery. Primary request: create a seamless modern stealth camouflage albedo texture befitting an aircraft called Wraith. Style/medium: realistic game-ready flat color texture, top-down orthographic, no perspective. Color palette: near-black charcoal base, deep gunmetal, cool slate grey, restrained blue-black; low-visibility and menacing. Composition: large angular fragmented polygon fields with some smaller faceted breaks, evenly distributed with no focal point. Materials/textures: subtle matte coating variation and extremely restrained worn scuff texture, but no baked lighting. Constraints: perfectly seamless on all four edges; flat albedo only; no highlights, shadows, bevels, rivets, panel lines, text, logos, insignia, symbols, aircraft parts, objects, borders, or watermark. Avoid: traditional green/brown camouflage, bright colors, glossy reflections, gradients caused by lighting, photoreal terrain, recognizable shapes.
